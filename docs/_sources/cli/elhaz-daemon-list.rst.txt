.. _elhaz-daemon-list:

``elhaz daemon list``
=====================

Synopsis
--------

.. code-block:: text

   elhaz daemon list

Description
-----------

List the names of all active sessions currently held in the daemon's session
cache, one per line.

The daemon must be running before calling this command.

Options
-------

``--help``
   Show help message and exit.

Examples
--------

.. code-block:: bash

   elhaz daemon list
