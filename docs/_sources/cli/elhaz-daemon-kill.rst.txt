.. _elhaz-daemon-kill:

``elhaz daemon kill``
=====================

Synopsis
--------

.. code-block:: text

   elhaz daemon kill

Description
-----------

Alias for :ref:`elhaz daemon stop <elhaz-daemon-stop>`. Sends a graceful
shutdown request to the running daemon and waits for it to exit.

Options
-------

``--help``
   Show help message and exit.

Examples
--------

.. code-block:: bash

   elhaz daemon kill
